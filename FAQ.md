# TerraPath FAQ

## Is TerraPath a wiki?

No. TerraPath is focused on structured player guides: progression stages,
loadouts, goals, alternatives, and short explanations.

## Can one guide support modded classes?

Yes. Guides use tags such as `melee`, `summoner`, `rogue`, or mod-specific tags.

## Will the website know every tModLoader item?

No. The web editor uses curated supported-content indexes. The in-game mod can
resolve content from installed mods.

## Why JSON instead of a free-form article?

Structured JSON can be validated, filtered, sorted, translated, cached, and shown
inside Terraria without parsing a forum-style page.
