# TerraPath Hub

TerraPath is a community guide platform for Terraria and tModLoader.

This repository hosts the public guide catalog, the web-based guide editor,
guide schemas, supported-content indexes, and community issue tracking.

## Project status

TerraPath is in early development. The public catalog and editor are being
prepared before the first in-game release.

## What TerraPath does

- Lets players browse class and progression guides in-game.
- Lets guide authors create structured guides instead of long forum posts.
- Supports weapons, armor, accessories, bosses, ores, materials, notes, and progression stages.
- Keeps guide data separate from the core mod so guides can update without a mod update.

## Public data

Community guides are stored as JSON files in `guides/`.

Generated catalog files are stored in `catalog/`.

Supported item, NPC, boss, ore, and icon indexes are stored in `supported/`.

## Private mod source

The tModLoader mod source is developed separately. This public hub is for the
community catalog, documentation, submissions, and the web editor.
